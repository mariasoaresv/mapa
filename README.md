# Um mapa de acessibilidade interativo, permitindo que usuários cadastrados colaborem e registrem pontos de atenção, obstáculos e rotas acessíveis em tempo real.


## Como Rodar o Projeto:

Pré-requisitos:
- Git
- Node.js

Guia de Instalação:
- Clone o repositório
  "git clone https://github.com/mariasoaresv/Mapa.git"
  
- Navegue até a pasta do projeto:
  "cd "caminho do repositorio" "
  
- Instale as dependências do Node.js: (Este comando vai baixar tudo que o server.js precisa para rodar)
  "npm install"
  
- Inicie o servidor: (Este comando executa o seu arquivo server.js)
  "node server.js"
  
- Abra seu navegador e acesse "http://localhost:3000"
